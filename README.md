# 🚀 Full-Stack Type-Safe CRUD (Vue 3.5 + Laravel 13)

A professional, type-safe CRUD application built with a focus on modular architecture and modern UI/UX principles. This project demonstrates seamless integration between **Vue 3.5 (TypeScript)** frontend and a **Laravel 13** backend.

## 🛠 Tech Stack
- **Frontend:** Vue 3.5 (Composition API + **TypeScript**)
- **Styling:** **Tailwind CSS v4** & Bootstrap Icons
- **Backend:** **Laravel 13** (REST API)
- **Database:** **MySQL** & **SQL Server**
- **Tooling:** Vite, NPM, Composer

## ✨ Key Features
- **Strict Type-Safety:** Interface-driven development for consistent data handling.
- **Modern UI:** Sticky headers and custom scrollable tables built with **Tailwind v4**.
- **Async UX:** Dynamic "Processing..." states and animated spinners during CRUD operations.
- **Optimistic Updates:** Local state filtering for immediate UI feedback after deletion.

## ⚙️ Installation & Setup

### 1. Database Setup (MySQL):-
1. Create a new database in **SQL Server** (e.g., `db_vue_crud`).
2. Import the **`db_vue_crud.sql`** file (found in the root folder) into your MySQL database to get the schema and sample data.

### 2. Backend (Laravel)
Navigate to the `backend` directory:

- composer install
- cp .env.example .env
# Update your DB_DATABASE, DB_USERNAME, and DB_PASSWORD in .env
php artisan key:generate
php artisan serve

### 3. Frontend (Vue 3 + TS):-
Navigate to the `frontend` directory:

npm install
npm run dev

### 📂 Technical Highlights:-
- **Modular Flow:** Decoupled business logic from UI components for better maintainability.
- **Clean Validation:** Hybrid validation using HTML5 types and Vue-side Regex for data integrity.
